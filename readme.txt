Easy Boats plugin provides boat post type and agent post type with related functionality.

== Description ==

Easy Boats plugin provides simple and easy way to built boat related websites.

It contains boat and agent custom post types and meta boxes settings to provide extensive options and features for your website.

This plugin offers only **one menu** to cover its settings and features. So, no trouble in finding where the settings are.

Major features in Easy Boats include:

* Manage boats types, hull types, statuses, locations and features.
* Ability to set boat as featured.
* Ability to set boat price, built year and length.
* Gallery Images.
* Boat Video
* Agent Information
* and many more...

== Installation ==

1. Unzip the downloaded package
1. Upload `easy-boats` to the `/wp-content/plugins/` directory
1. Activate the `Easy Boats` through the 'Plugins' menu in WordPress
